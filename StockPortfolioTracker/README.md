# Stock Portfolio Tracker

This is a simple Python project that tracks a stock portfolio using hardcoded stock prices.

## Features

- User enters stock symbols and quantities
- Uses a Python dictionary for stock prices
- Calculates total investment value
- Displays a portfolio summary
- Optionally saves the result as `.txt` or `.csv`

## Stock Prices Used

- `AAPL`: 180
- `TSLA`: 250
- `GOOGL`: 140
- `MSFT`: 330
- `AMZN`: 145

## How to Run

```bash
python stock_portfolio_tracker.py
```

## Example Flow

1. Enter a stock symbol like `AAPL`
2. Enter the quantity
3. Repeat for more stocks
4. Type `done` to finish
5. Choose whether to save the summary
